# 飞书App机器人创建指南（2024版）

## 问题：找不到「自定义」选项

### 原因分析
1. **版本差异**：飞书App不同版本界面布局不同
2. **权限限制**：个人账号可能无法创建机器人
3. **功能迁移**：部分功能已整合到「智能助手」中

### 解决方案

## 方案1：使用「智能助手」（推荐）
1. 打开飞书App → 进入任意聊天界面
2. 点击右下角「+」→ 选择「智能助手」
3. 选择「创建新助手」
4. 设置助手名称和描述
5. 点击「生成」
6. 系统会自动创建一个AI助手，可发送消息

## 方案2：通过网页版创建
1. 打开浏览器 → 访问 https://www.feishu.cn/
2. 登录您的飞书账号
3. 进入「工作台」→ 搜索「机器人」
4. 点击「创建机器人」→ 选择「自定义机器人」
5. 填写信息 → 获取Webhook URL

## 方案3：使用企业微信替代
如果飞书无法满足需求，可考虑使用企业微信：
1. 下载企业微信App
2. 创建群聊 → 添加机器人
3. 获取Webhook URL → 使用相同代码

## 注意事项
- 个人账号权限有限，建议使用企业账号
- 「智能助手」功能更强大，支持AI对话
- Webhook URL需妥善保管，避免泄露

## 联系客服
如仍无法创建，请联系飞书客服：https://help.feishu.cn/hc/zh-cn
